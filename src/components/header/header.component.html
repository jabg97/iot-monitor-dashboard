About

This is an educational project to monitor crops through an IoT system deployed in Azure cloud.

This project was developed using Typescript, Azure Functions and Cosmo DB for backend programming and Typescript, Angular, SCSS and HTML for frontEnd programming.

A variant of the C++ programming language was also used for the programming of the Arduino microcontroller.

All of this in order to obtain a degree in Informatic Engineering.
